# Crypto Market ETL & Analysis

## Overview
An end-to-end ETL pipeline that collects cryptocurrency market data
(price, volume, market cap, sentiment) and prepares it for analysis.

## Data Sources
- CoinGecko API
- Crypto Fear & Greed Index

## Architecture
Extract → Transform → Load

## Metrics Collected
- Price (OHLC)
- Trading Volume
- Market Cap
- Fear & Greed Index
- Volatility (derived)

## Tech Stack
- Python
- Pandas
- SQLite
- REST APIs
